# 異環空幕裝配 Solver

這是一個用來輔助《異環》空幕裝配的方塊擺放求解器。

它可以讓你手動設定不規則盤面、黑格、固定方塊，以及剩餘可重複使用的 2 型、3 型、4 型方塊，然後列出可行的裝配結果。

## 功能

- 自訂盤面大小，預設為 4 x 5。
- 可把格子標成空格或黑色不可放。
- 支援固定方塊：
  - 可指定形狀但不指定位置，求解時必須使用。
  - 也可拖曳放到指定位置。
  - 固定槽可用 `+` / `-` 增減。
- 支援剩餘方塊多選，方塊可重複使用。
- 方塊方向固定，不旋轉、不鏡像。
- 支援補滿模式與可空 1 格模式。
- 可匯入/匯出完整設定，包含盤面、固定方塊、剩餘方塊與求解設定。
- 可匯出解答 JSON。
- 內建角色/套裝預設固定方塊清單，可在程式中補上對應的方塊 ID。

## 執行

### Python

```bash
python yihuan_tiling_solver.py
```

### Windows exe

也可以直接執行：

```text
yihuan_tiling_solver.exe
```

## 預設固定方塊

預設固定方塊清單在 `yihuan_tiling_solver.py` 的 `PRESET_FIXED_PIECES`。

填法範例：

```python
{"id": "lost_radiance", "name": "失落光芒", "en": "Lost Radiance", "fixed_shape_ids": ["3-1", "2-2", "4-5", "3-4"]},
```

`fixed_shape_ids` 的順序對應固定1、固定2、固定3、固定4，以此類推。

可用 ID 就是介面中看到的 `2-1`、`2-2`、`3-1`、`4-19` 等方塊 ID。
