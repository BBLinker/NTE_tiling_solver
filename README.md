# 異環空幕裝配 Solver

這是一個用來輔助《異環》空幕裝配的方塊擺放求解器。

它可以讓你手動設定不規則盤面、黑格、固定方塊，以及剩餘可重複使用的 2 型、3 型、4 型方塊，然後列出可行的裝配結果。

## 功能

- 自訂盤面大小，預設為 4 x 5。
- 可把格子標成空格或黑色不可放。
- 支援固定方塊：
  - 可指定形狀但不指定位置，求解時必須使用。
  - 也可拖曳放到指定位置。
  - 固定槽可用 `+` / `-` 增減。
- 支援剩餘方塊多選，方塊可重複使用。
- 方塊方向固定，不旋轉、不鏡像。
- 支援補滿模式與可空 1 格模式。
- 可匯入/匯出完整設定，包含盤面、固定方塊、剩餘方塊與求解設定。
- 可匯出解答 JSON。
- 內建 12 組卡帶與 14 個角色的預設固定方塊。

## 執行

### Python

```bash
python yihuan_tiling_solver.py
```

### Windows exe

Windows 版執行檔會放在 GitHub Releases，不直接放在 repository 裡。

下載 Release 附件後可直接執行：

```text
yihuan_tiling_solver.exe
```

如果要自行重新打包，可使用 repository 內的 `yihuan_tiling_solver.spec` 搭配 PyInstaller。

## 預設固定方塊

程式已內建 12 組卡帶與 14 個角色的預設固定方塊。

在左下角的「預設固定方塊」區塊中點選預設項目，即可自動套用對應的固定方塊設定。套用後仍可在固定方塊區手動調整，手動調整後會取消目前預設的選取狀態。
